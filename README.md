# 🐦 Yapper — Um Microblog estilo Twitter com Django + (React talvez :D )

Bem-vindo ao **Yapper**, um projeto acadêmico desenvolvido para a disciplina de **Programação Web** no IFMT Campus Octayde.  
Este microblog permite postagens curtas estilo Twitter e está sendo construído com boas práticas usando Django — e com frontend em React em fase de estudo e possível implementação.

---

## 🚀 Sobre o Projeto

O **Yapper** é uma rede social minimalista onde qualquer usuário pode:

- ✨ Criar uma conta
- 🔐 Fazer login/logout com segurança
- 📝 Postar mensagens curtas (Yaps)
- 🧭 Navegar por uma timeline de Yaps

A ideia é aplicar os conhecimentos de Django na criação de um sistema completo com autenticação, banco de dados, CRUD e organização modular.
